# Diagramas de Sequência - Sistema de Moeda Estudantil

Arquivos em PlantUML corrigidos para exibir as barras/caixas de duração das atividades nas lifelines.

Foi adicionado `autoactivate on` em todos os diagramas, além de parâmetros visuais para deixar as ativações mais visíveis.

## Arquivos

01. realizar autenticação/login
02. cadastrar aluno
03. selecionar instituição de ensino
04. cadastrar empresa parceira
05. cadastrar vantagem
06. resgatar vantagem
07. gerar código do cupom
08. enviar cupom por e-mail
09. consultar extrato da conta
10. enviar moedas de mérito
11. notificar aluno por e-mail
12. informar motivo do envio
13. adicionar 1.000 moedas semestralmente
